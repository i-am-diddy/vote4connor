# vote4connor
## Installation

1. **Install a Userscript Manager**
   - [Tampermonkey](https://www.tampermonkey.net/) (Chrome, Firefox, Edge, Safari)
   - [Violentmonkey](https://violentmonkey.github.io/) (Chrome, Firefox)

2. **Install the Script**
   - Install from [Greasyfork](https://greasyfork.org/en/scripts/566039-connor-gutierrez-auto-voter)
   - Navigate to [Voting Page](https://www.elpasotimes.com/story/sports/high-school/2026/02/10/vote-el-paso-childrens-hospital-hs-male-athlete-for-feb-3-7/88578099007/?fbclid=PAZXh0bgNhZW0CMTEAc3J0YwZhcHBfaWQMMjU2MjgxMDQwNTU4AAGnhpAr_k0H-zpccZUMsYxSMqo9XMd4y00HE9DUpoO6QF8UmrVxr33PGtwVZLo_aem_Jf-R6UBtXB7qN3GiXKj5BA)
   - Wait and watch
